⚠️ Important: Fix CORS Errors

Your website needs to run on a local web server

❌Current Problem

You're seeing these errors in the browser console:

  • Access to script blocked by CORS policy
  • Failed to load resource: net::ERR_FAILED
  • Failed to load module script

Why? You're opening the HTML file directly (file:// protocol). Modern browsers block JavaScript modules for security when loaded this way.

✅The Solution

Run a local web server! Here's the easiest way:

1 Open Command Prompt
  • Press Win + R
  • Type cmd and press Enter
2 Navigate to your project folder
cd C:\Users\YourName\Downloads\ecommerce-main
Replace with your actual path
3 Start the server
python -m http.server 8000
Or double-click START-SERVER.bat
4 Open your browser to:
http://localhost:8000
⚡ Don't have Python?

Download it from: python.org/downloads

During installation, make sure to check "Add Python to PATH"
