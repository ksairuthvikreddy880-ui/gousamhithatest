Step 4: Permissions Setup Instructions

⚠️ You MUST set permissions in Nhost Console:

  1. Go to: Orders Table
  2. Click "Permissions" tab
  3. Add or edit "user" role
  4. For SELECT permission:
    • Row select permissions: {"customer_id":{"_eq":"X-Hasura-User-Id"}}
    • Column permissions: Select all
  5. Click Save
  6. Repeat for order_items table with: {"order":{"customer_id":{"_eq":"X-Hasura-User-Id"}}}
