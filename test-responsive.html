Testing Instructions

How to Test:

  1. Desktop Browser: Open DevTools (F12) and use Device Toolbar
  2. Test Widths: 375px (iPhone), 768px (iPad), 1024px (Laptop), 1440px (Desktop)
  3. Real Devices: Test on actual phones and tablets
  4. Orientation: Test both portrait and landscape modes
  5. Touch: Verify all buttons and links are easily tappable
  6. Scroll: Ensure no horizontal scrolling occurs

Pages to Test:

  • Home Page
  • Shop Page
  • Cart Page
  • Checkout Page
  • Orders Page
  • Profile Page
  • About Page
  • Contact Page
